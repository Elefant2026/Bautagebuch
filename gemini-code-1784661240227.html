Möchtest du deinen Baufortschritt mit deinen Liebsten teilen?

Möglichkeiten zum Hosten:

  • Kostenlos & Einfach (GitHub Pages / Netlify / Vercel): Lade diese Datei auf Plattformen wie GitHub oder Netlify hoch, um eine eigene Online-Adresse (z. B. unsere-sanierung.netlify.app) zu erhalten.
  • Eigener Webspace: Falls du bereits eine Domain hast, lade die Datei einfach per FTP in deinen Webspace hoch.
  • Passwortschutz: Über deinen Webhosting-Anbieter kannst du einen Verzeichnisschutz (.htaccess) einrichten, damit nur Familie & Freunde mit Passwort Zugriff haben.
